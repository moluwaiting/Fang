### landslide使用教程
-------------------------
* 源码安装landslide
```shell
$ git clone https://github.com/adamzap/landslide.git
$ cd landslide
$ python setup.py build
$ sudo python setup.py install
```
* markdown格式转化为html
```shell
$ landslide README.md -i -o > readme.pdf
```
* markdown格式转化为pdf
```shell
$ landslide README.md -d readme.pdf
$ open readme.pdf
```
